# Attack Surface Management (ASM) Platform

## Overview

Attack Surface Management (ASM) Platform is a Python-based cybersecurity tool designed to discover and analyze an organization's external attack surface.

## Features

- Subdomain Discovery
- Live Host Detection
- Port Scanning
- Technology Detection
- Security Header Analysis
- SSL Certificate Analysis
- Risk Scoring
- PDF Report Generation

## Technologies Used

- Python
- Nmap
- Subfinder
- Httpx
- SQLite
